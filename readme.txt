Generate recipes based on ingredients you have on hand for melodyskitchen.com.

== Description ==
Melody's Recipe Generator allows users to input ingredients they have and suggests recipes using a custom database or an external API (e.g., Spoonacular). Perfect for melodyskitchen.com!

== Installation ==
1. Upload the `melodys-recipe-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the `[recipe_generator]` shortcode to any page or post.
